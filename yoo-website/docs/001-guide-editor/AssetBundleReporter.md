# 构建报告

报告工具，可以查看概览信息（Summary），资源对象列表信息（AssetView），资源包列表信息（BundleView）。

**注意**：该工具仅支持Unity2019.4+

### 资源对象列表视图

可以查看资源对象列表以及每个资源对象所依赖的资源包。

![image](./Image/AssetReporter-img2.png)

### 资源包列表视图

可以查看资源包列表以及每个资源包所包含的资源对象。
