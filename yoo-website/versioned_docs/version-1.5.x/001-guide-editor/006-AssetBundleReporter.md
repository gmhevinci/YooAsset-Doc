# 构建报告

学习如何查看资源构建报告。

报告工具，可以查看概览信息（Summary），资源对象视图，资源包视图。

**注意**：该工具仅支持Unity2019.4+

### 资源对象列表视图

![image](./Image/AssetReporter-img2.jpg)
